# OnlyofficeLoggerHelper

Helper gem for logging info messages in Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'onlyoffice_logger_helper'
```

And then execute:

```shell script
bundle
```

Or install it yourself as:

```shell script
gem install onlyoffice_logger_helper
```
